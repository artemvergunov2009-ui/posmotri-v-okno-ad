# Проектная работа «Посмотри в окно» 🪟

В этом проекте отрабатываются навыки написания CSS-стилей для уже готового приложения.
Стартовый репозиторий изначально содержит HTML-разметку, файлы шрифтов и JavaScript-логику, которая динамически подставляет данные с использованием тегов `<template>`.

## Задачи проекта:
1. **Лейаут:** Реализация сеток с помощью CSS Grid и Flexbox (`.page` и `.content`).
2. **Типографика и размеры:** Использование свойст `object-fit`, `line-clamp`, `flex-grow`.
3. **Кастомные формы:** Стилизация инпутов, сброс дефолтных стилей браузера через `appearance: none`, стилизация кастомных радио-кнопок и чекбоксов (`.visually-hidden`).
4. **Интерактивные состояния:** Проработка состояний `:hover`, `:active`, `:focus` и `:focus-visible` (с использованием селектора `:has()`).
5. **Позиционирование прелоадеров:** Настройка абсолютного позиционирования поверх нужных относительных контейнеров.

## Технологии:
* HTML5 (семантика, `template`)
* CSS3 (Grid, Flexbox, Custom properties)
* Vanilla JavaScript (DOM-манипуляции для мок-данных)

## Запуск
Для запуска достаточно открыть файл `index.html` в браузере или использовать расширение типа *Live Server* в VS Code.
